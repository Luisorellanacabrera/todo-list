# ✅ Lista de Tareas (To-Do List)

Aplicación de lista de tareas creada con **HTML, CSS y JavaScript puro**.

## 🎯 Descripción
Proyecto del Bootcamp Full Stack que permite gestionar tareas personales con:
- ➕ Agregar nuevas tareas
- ✔️ Marcar tareas como completadas
- ❌ Eliminar tareas
- 🔍 Filtrar por estado (todas / pendientes / completadas)
- 💾 Guardado automático con **localStorage**

## 🛠️ Tecnologías usadas
- HTML5
- CSS3 (sin frameworks)
- JavaScript (vanilla)
- localStorage (persistencia de datos)

## ▶️ Cómo usar
1. Abre `index.html` en el navegador
2. Escribe una tarea en el campo de texto
3. Presiona **Agregar** o **Enter**
4. Haz clic en el círculo para marcarla como completada
5. Usa los filtros para ver distintas vistas
